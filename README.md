# HTML Authentication POC

This project is a Proof of Concept (POC) for a basic web authentication flow using only plain HTML.

## Pages Included:
1. **Login** (`login.html`)
2. **Register** (`register.html`)
3. **Forgot Password** (`forgot-password.html`)
4. **Reset Password** (`reset-password.html`)
5. **Dashboard** (`dashboard.html`)

## Instructions:
Open `login.html` in any browser to navigate through the flow using the provided anchor tags.
